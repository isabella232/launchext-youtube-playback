How this Event works

This Event is triggered when the YouTube player has finished playing a video.

The following properties are available in the event data element. You can use any of them with your Rule's Conditions and Actions.

event field Data type Value
event.state String Fixed value: video ended
event.youtube.player JavaScript object The YouTube player object.
event.youtube.currentTime Floating point number Elapsed time in seconds since the video started playing.
event.youtube.duration Integer Duration in seconds of the currently playing video.
event.youtube.muted Boolean Whether the video is muted (true) or unmuted (false).
event.youtube.playbackRate Floating point number Playback rate of the currently playing video.
event.youtube.videoLoadedFraction Floating point number Percentage of the video that the player shows as buffered.
event.youtube.videoUrl String YouTube.com URL for the currently loaded/playing video.
event.youtube.volume Integer The player's current volume.
